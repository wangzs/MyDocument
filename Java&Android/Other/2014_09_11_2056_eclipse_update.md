###Eclipse的运行加速配置
**eclipse.ini**
```
-Xverify:none
-Xmx1024m
-Xms1024m
-Xmn512m
-XX:PermSize=512m
-XX:MaxPermSize=512m
-XX:+DisableExplicitGC
-Xnoclassgc
-XX:+UseParNewGC
-XX:+UseConcMarkSweepGC
-XX:CMSInitiatingOccupancyFraction=85
```
